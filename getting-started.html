OpenAperture

Cloud Application Management Platform

Products that are being developed for deployment into the Cloud should follow the guidance provided by The 12 Factor App. As a result of how OpenAperture deploys products, there are several design constraints that should be considered:

  • Products must be able to scale horizontally.
  • Products must be able to accept configuration via environment variables (i.e. database URLs, etc...).
  • Products must be dockerizable. Currently, this requires products to run in a Linux environment.
