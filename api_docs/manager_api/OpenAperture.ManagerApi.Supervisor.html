(function) # ↑

Specs:

  • start_link :: {:ok, pid} | {:error, String.t}

Method to start the supervisor as a process tied to the current supervision tree

Return values

If the supervisor is successfully created and initialized, the function returns {:ok, pid}, where pid is the pid of the supervisor. If there already exists a process with the specified supervisor name, the function returns {:error, {:already_started, pid}} with the pid of that process.

If the init/1 callback fails with reason, the function returns {:error, reason}. Otherwise, if it returns {:stop, reason}

or :ignore, the process is terminated and the function returns {:error, reason} or :ignore, respectively.
