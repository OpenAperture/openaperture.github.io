erlang_timestamp_to_unix_timestamp(erlang_timestamp)
(function) # ↑

Specs:

  • erlang_timestamp_to_unix_timestamp(OpenAperture.Router.Types.erlang_timestamp) :: OpenAperture.Router.Types.unix_timestamp

Converts an erlang timestamp to a unix timestamp. Erlang timestamps are of the form {MegaSecs, Secs, MicroSecs}, of the elapsed time since the start of the Unix epoch. See http://erlang.org/doc/man/erlang.html#now-0 for more details on Erlang timestamps. Unix timestamps are an integer representing the number of seconds since the start of the Unix epoch.

This conversion is necessarily lossy, since Unix timestamps have only one second resolution, while Erlang timestamps have microsecond resolution.
