Specs:

  • parse_hostspec(String.t) :: {:ok, String.t, integer} | :error

Parses a string of the form hostname:port into its separate hostname and port components. Supports URL-encoded colons.

Example

iex> OpenAperture.Manager.Controllers.Router.Util.parse_hostspec("test:80")
{:ok, "test", 80}

iex> OpenAperture.Manager.Controllers.Router.Util.parse_hostspec("test%3A80")
{:ok, "test", 80}

iex> OpenAperture.Manager.Controllers.Router.Util.parse_hostspec("test%3a80")
{:ok, "test", 80}
