OpenAperture.Builder
OpenAperture.Builder.BuildLogPublisher

BuildLogPublisher tracks a list of current exchange_id’s, which is cleared and recreated periodically. Additionally, it tracks a hashdict of ExchangePublisher Genserver pids, which is only ever appended to. When publish_build_logs is called, it sends the logs to the ExchangePublisher for each entry in the current exchange list
