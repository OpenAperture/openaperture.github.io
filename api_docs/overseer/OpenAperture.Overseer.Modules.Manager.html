This module contains the GenServer for managing Module communication processes

Summary↑

handle_cast(msg, state)

GenServer callback for handling the :set_modules event. This method will update the module list and stop/start listeners are needed
inactivate_listeners(modules)

Method to identify and update or delete any "inactive" listeners. A listener is defined as "inactive" if: * The last updatedat time is > 10 minutes - set status to :inactive * The last updatedat time is > 20 minutes - delete reference

Method to set the known list of system modules into the manager
